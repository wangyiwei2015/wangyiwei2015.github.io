Windows 弹出 USB 设备失败时的处理

  2022-05-19   Windows   技巧 阅读时长: 1分

在 Windows 中弹出 USB 存储设备时，可能会遇到 该设备正在使用中，请关闭可能使用该设备的所有程序或窗口 。

此时，有两种方法检查占用磁盘的程序或进程。

资源监视器

运行 perfmon.exe 或者 | 开始 => 输入 perfmon

在 CPU 选项卡下，句柄搜索目标盘符（例 F: ），找到相应的占用者，杀之。

事件查看器

运行 eventvwr.msc 或者 | 开始 => 输入 eventvwr

在左侧选中 Windows 日志 的 系统 ，找到 警告 级别且来源为 Kernal-PnP 的事件，会发现 XXX 已停止删除或弹出设备 USB\XXX ，所指出的应用程序就是占用者，杀之。

更方便的长期使用技巧：在左侧 自定义视图 上右击， 创建自定义视图 ，填写以下配置：

  • 记录时间：选择 近一小时
  • 事件级别：选中 警告
  • 按源：选中
  • 事件来源：Kernal-PnP

以后就可以通过自定义视图查找。
